Introducción al lenguaje

También conocido como Hyper Text Markuo Languaje o Lenguaje de Marcado de Hipertexto. Es el fundamento o lenguaje sobre el que se construye toda página web,
se utiliza para estructurar el contenido de forma sencilla para que cualquier usuario pueda acceder a la información de forma sencilla y útil.

Etiquetas de Formato

La etiqueta < hr > se utiliza en HTML para insertar una regla horizontal o crear una separación temática entre secciones de contenido en una página web. Ayuda a mejorar la estructura visual de tu sitio web.
Por otro lado, La etiqueta < br > representa el mencionado salto de línea, para avanzar a la línea siguiente y continuar escribiendo en una nueva línea, pero llevado a un documento web, ya que un ENTER no realizará esa acción en el documento renderizado en el navegador.
Tanto las etiquetas < br > (salto de línea) como < hr > (línea horizontal) pueden crear espacio entre elementos, cumplen funciones diferentes. La etiqueta < br > mueve el contenido a la siguiente línea, mientras que la etiqueta < hr > crea una ruptura temática en el contenido con una línea horizontal.
